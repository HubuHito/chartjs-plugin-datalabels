# 脚本配置项

脚本配置项支持函数，每个数据集都会调用该函数，并提供 context 参数（查看 [context]）

例子：

color: function(context) {
    var index = context.dataIndex;
    var value = context.dataset.data[index];
    return value < 0 ? 'red' :  // draw negative values in red
        index % 2 ? 'blue' :    // else, alternate values in blue and green
        'green';
}

# Option Context

Context 上下文对象提供解析数据的信息，主要用于 脚本配置项，也适用函数类型的配置，如formatter。

上下文对象提供如下属性：

属性名 类型 描述
active bool 元素是否处于悬浮状态 (查看 interactions (opens new window)).
chart Chart 图表对象
dataIndex number 当前数据项的索引
dataset object datasetIndex 索引处对应的数据集
datasetIndex number 数据集索引

# Indexable Options

Indexable Options 类型的属性支持数组类型，数组中的每一项和数据项一一对应。注意：数组的长度需要和数据项的长度保持一致，大多数情况下，推荐使用 函数

例子：

color: [
    'red',    // color for data at index 0
    'blue',   // color for data at index 1
    'green',  // color for data at index 2
    'black',  // color for data at index 3
    //...
]

# Style Options

Style options 类型通常用于 fillStyle (opens new window) or strokeStyle (opens new window) 这样的属性。

支持以下类型的值：

  • CSS 颜色 (opens new window)
  • CanvasGradient (opens new window) 对象 (线性和径向渐变)
  • CanvasPattern (opens new window) 对象

例子：

color: 'green'                  // named color
color: '#dc143c'                // HEX color
color: 'rgb(51, 170, 51)'       // RGB color (opaque)
color: 'rgba(51, 170, 51, .5)'  // RGBa color (semi-transparent)
// ...
