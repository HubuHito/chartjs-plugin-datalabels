# 介绍

可高度自定义的 Chart.js (opens new window) 的文本标签插件，支持任意图表类型。

兼容性

支持 Chart.js (opens new window) 3.0.0-beta.7。

重要

插件不提供公有API（除了插件钩子函数），因此最好不要调用 $ 或者 _ 开头的属性和方法，包括 $datalabels 插件的属性。在后面的版本这些私有方法可能会被删除。
